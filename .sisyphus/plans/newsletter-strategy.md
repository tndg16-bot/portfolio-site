# ニュースレター配信戦略

## 📌 基本方針

| 項目 | 内容 |
|------|------|
| **配信頻度** | 月2回（月初・月中目安） |
| **ターゲット** | 自分を変えたい人 |
| **最終ゴール** | セッション予約につなげる |
| **CTA** | 記事を読む → 記事内でセッション誘導 |

---

## 📧 配信コンテンツ構成

### 配信1: メインニュースレター（月1回）
**内容の濃いオリジナルコンテンツ**

構成例:
- 今月のテーマ / メッセージ
- 深掘りコラム（記事にはしない限定内容）
- 実践ワーク / 問いかけ
- おすすめ記事ピックアップ
- 近況報告 / ポートフォリオ進捗

### 配信2: ライト配信（月1回）
**データ連携 or 自動生成系**

候補:
- [ ] 新着記事まとめ（自動取得）
- [ ] 今月のアクセスランキング
- [ ] SNS投稿ハイライト
- [ ] 読者からの質問 & 回答（Q&A形式）

---

## 🎁 登録特典（要検討）

登録率を上げるためのインセンティブ。

### 候補案

| 特典 | メリット | デメリット | 難易度 |
|------|----------|------------|--------|
| **PDFガイド** | 作り込める、保存される | 作成に時間かかる | 中 |
| **チェックリスト** | 実用的、すぐ使える | 薄く見える可能性 | 低 |
| **限定記事** | コンテンツ資産になる | 継続的に出すのは大変 | 中 |
| **動画コンテンツ** | 差別化できる | 制作コスト高 | 高 |
| **無料相談（15分）** | 最も効果的、直接つながる | 時間コスト | 低〜中 |
| **自己分析テンプレート** | ターゲットに刺さる | 内容次第 | 低〜中 |

### 決めるべきこと
- [ ] どの形式にするか（PDF / チェックリスト / 動画 等）
- [ ] テーマは何にするか
- [ ] ボリューム感（1ページ / 10ページ）
- [ ] 作成期限

### 特典テーマ案（ターゲット「自分を変えたい人」向け）
1. 「自分を変える最初の7日間」ガイド
2. 「本当の自分を見つける」自己分析ワークシート
3. 「行動できない自分を卒業する」チェックリスト
4. 「AI時代のキャリア棚卸し」テンプレート
5. 「モヤモヤ整理」ジャーナリングテンプレート
